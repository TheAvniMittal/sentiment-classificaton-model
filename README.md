Developed TextEmotionAI, an end-to-end NLP project that classifies emotions from text using machine learning.
Performed text preprocessing including lowercasing, punctuation removal, stopword removal, and tokenization using NLTK. 
Trained and compared multiple models including Naive Bayes and Logistic Regression with TF-IDF and Bag of Words vectorization.
The final model — Logistic Regression with BoW — achieved 88.9% accuracy across 6 emotion classes. Deployed as a real-time interactive 
web application using Streamlit with confidence scores and probability breakdown for each prediction.
